# Retail Sales Management System

## Overview
A complete Retail Sales Management database built using SQL Server Management Studio (SSMS).

## Database Structure
- 10 interlinked tables: Customers, Orders, Products, Inventory, Employees, Branches, Payments and more
- Normalized relational database design with primary and foreign keys

## What's Included
- 30+ SQL queries (JOINs, Subqueries, GROUP BY, Window Functions)
- 5 Stored Procedures including dynamic product search
- Scalar and Table-Valued Functions (discount, tax, order history)
- Transaction Management (COMMIT, ROLLBACK, SAVEPOINT)
- 4 Views for simplified reporting

## Tools Used
- SQL Server Management Studio (SSMS)
- T-SQL

## Key Learnings
- Database design and normalization
- Writing complex queries for business reporting
- Transaction handling to prevent data errors
